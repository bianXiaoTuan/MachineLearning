训练集包含项
* PassengerId: 乘客编号
* Survived: 是否幸存
* Pclass: 乘客级别((1 = 1st; 2 = 2nd; 3 = 3rd), 体现乘客社会地位
* Name: 名字
* Sex: 性别
* Age: 年龄
* SibSp: 船上兄弟姐妹/配偶的人数
* Parch: 船上父母/子女的人数
* Ticket: 船票号码
* Fare: 旅客票价
* Cabin: 船舱
* Embarked: 上船港口(C = Cherbourg; Q = Queenstown; S = Southampton)
